# Common ports

#### Port 21

Service: File Transfor Protocol (FTP)
Protocol: TCP
Description: Used to transfer files from host to host.

#### Port 22

Service: SSH, SCP, and SFTP
Protocol: TCP
Description: Provides secure remote termnial access and file transfer capabilities, provides secure copy functions, provides secure file trnasfers

#### Port 23

Service: Telnet
Protocol: TCP
Description: Provides insecure remote controll of another machine using a text-based environment

#### Port 25

Service: Simple Mail Transfer Protocol (SMTP)
Protocol: TCP
Description: Used for sending emails over the network.

#### Port 53

Service: Domain name system (DNS)
Protocol: TCP/UDP
Description: Translates domin names into IP addresses

#### Port 69

Service: Trivial file transfer protocol (TFTP)
Protocol: UDP
Description: Used for lightweight file transfers like config files, or network booting of an OS
    
#### Port 80

Service: HTTP
Protocol: TCP
Description: Used for retrieving web pages in an unencrypted format.

#### Port 88

Service: Kerberos
Protocol: UDP
Description: Provides network authentication.


#### Port 110

Service: Post office protocol version Three (POP3)
Protocol: TCP
Description: Retrieves email from a server.

#### Port 119

Service: Network news transfer protocol (NNTP)
Protocol: TCP
Description: Used for accessing newsgroups.

#### Port 135

Service: Remote procedure call (RPC)
Protocol: TCP/UDP
Description: Facilitates communication between different system processes.

#### Ports 137, 138, and 139

Service: NetBIOS
Protocols: TCP/UDP
Description: Networking protocol seite.

#### Port 143

Service: Internet message access protocol (IMAP)
Protocol: TCP
Description: Allow accesses to email messages on a server.

#### Port 161

Service: Simple network management protocol (SNMP)
Protocol: UDP
Description: Manages network devices.

#### Port 162

Service: SNMP Traps
Protocol: UDP
Description: Sends SNMP trap messages.

#### Port 389

Service: Lightweight directory access protocol (LDAP)
Protocol: TCP
Description: Used for directory services.

#### Port 443

Service: HTTPS
Protocol: TCP
Description: Provides secure web communication using encryption.

#### Port 445

Service: Server message block (SMB)
Protocol: TCP
Description: Used for file and printer sharing over a network.

#### Port 465 and 587

Service: SMTP Secure (SMTPS)
Protocol: TCP
Description: Secure SMTP communications.

#### Port 514

Service: Syslog
Protocol: UDP
Description: Sending log messages.

#### Port 636

Service: LDAP secure (LDAPS)
Protocol: TCP
Description: LDAP communication over SSL/TLS.

#### Port 993

Service: Internet message access protocol over SSL/TLS (IMAPS)
Protocol: TCP
Description: User for secure email retrieval.

#### Port 995

Service: Post office protocol version three over SSL/TLS (POP3S)
Protocol: TCP
Description: Used for secure email retrieval.

#### Port 1433

Service: Microsoft SQL
Protocol: TCP
Description: Used to facilitate communication with Microsoft SQL server.

#### Port 1645 and 1646

Service: RADIUS TCP
Protocol:TCP
Description: Used for remote authentication, authorization, and accounting.

#### Port 1812 and 1813

Service: RADIUS UDP
Protocol: UDP
Description: Used for authentication and accounting as defined by the internet engenering task force (IETF).

#### Port 3389

Service: Remote desktop protocol (RDP)
Protocol: TCP
Description: Enable remote desktop access.

#### Port 6514

Service: Syslog TLS
Protocol: TCP
Description: Used in a secure sylog that uses SSL/TLS to encrypt ip packets using certs, before sending the log to the log collector.
